## ‚ö° Quick Setup

We provide a `setup.sh` script to automate the installation of Homebrew and apply reasonable system defaults (Finder, Dock, and Interface settings).

### Usage

1.  Clone the repository:
    ```bash
    git clone https://github.com/kamalsoft/macOS-cheat-sheet.git
    cd macOS-cheat-sheet
    ```

2.  Make the script executable:
    ```bash
    chmod +x setup.sh
    ```

3.  Run the script:
    ```bash
    ./setup.sh
    ```
